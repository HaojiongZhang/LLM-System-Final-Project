# MiniTorch Attention Project

This repository contains MiniTorch implementations and experiments for FlashAttention and PagedAttention.

Branches:
- `main`: integrated FlashAttention + PagedAttention code used for the final report.
- `integrated-attention`: development branch for the combined FlashAttention and PagedAttention path.
- `paged-attention`: focused PagedAttention KV-cache implementation, tests, and benchmark.
- `fforward`: FlashAttention forward implementation.
- `flash_attention_2_backward_kevin`: FlashAttention backward implementation.
